<h1>Suppression du compte VBE</h1>

<p>Les utilisateurs peuvent supprimer leur compte directement depuis l'application VBE.</p>

<h2>Étapes</h2>
<ol>
    <li>Ouvrir l'application VBE</li>
    <li>Accéder à « Réglages »</li>
    <li>Sélectionner « Supprimer mon compte »</li>
    <li>Confirmer la suppression</li>
</ol>

<h2>Données supprimées</h2>
<ul>
    <li>Profil utilisateur</li>
    <li>Informations personnelles enregistrées</li>
    <li>Préférences associées au compte</li>
</ul>

<p>Certaines données peuvent être conservées temporairement lorsque la loi l'exige.</p>

<h2>Assistance</h2>
<p>Pour toute demande d'assistance concernant la suppression de compte, contactez l'équipe VBE via l'application.</p>
